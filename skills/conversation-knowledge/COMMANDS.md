### 3️⃣ 加载话题 Load Topic

| 模式 | 命令 | 示例 |
|------|------|------|
| **英文** | `/load Topic [format]` | `/load AI Agents summary` |
| **中文** | `/加载 话题 [格式]` | `/加载 AI 智能体分类 summary` |
| **自然语言** | - | "加载 AI 智能体话题"、"打开 XXX 讨论" |

**参数**:
- `Topic` - 话题名称（必填）
- `format` - 格式：`full`（默认）、`summary`、`timeline`

**功能**: 
- 加载指定话题的对话记录
- **自动设置上下文**（1 小时内有效）
- 后续 `/record` 会自动使用该话题

---

### 4️⃣ 记录对话 Record

| 模式 | 命令 | 示例 |
|------|------|------|
| **英文** | `/record --summary "..."` | `/record --summary "Discussed AI"` |
| **中文** | `/记录 --summary "..."` | `/记录 --summary "讨论了 AI"` |
| **自然语言** | - | "记一下"、"保存刚才的对话" |

**带上下文**（推荐）:
```bash
/加载 AI 智能体分类
/记录 --summary "讨论了新的分类方法"  # 自动记录到 AI 智能体分类
```

**手动指定**:
```bash
/记录 --topic "新话题" --summary "摘要"
```

**强制新建**:
```bash
/记录 --topic "AI" --summary "新讨论" --new
```

### 9️⃣ 话题管理 Manage Topics

| 模式 | 命令 | 示例 |
|------|------|------|
| **英文** | `/topics --action list` | `/topics list` |

## 📊 参数详解

### 通用参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--topic` | 话题名称 | `--topic "AI 智能体"` |
| `--summary` | 摘要内容 | `--summary "讨论了 XXX"` |
| `--channel` | 渠道 | `--channel "dingtalk"` |
| `--format` | 格式 | `--format "markdown"` |
| `--days` | 天数 | `--days 7` |
| `--limit` | 数量限制 | `--limit 10` |
| `--new` | 强制新建 | `--new` |

### 格式选项

| 格式 | 说明 | 使用场景 |
|------|------|----------|
| `full` | 完整对话记录 | 查看详情 |
| `summary` | 只看摘要 | 快速回顾 |
| `timeline` | 时间线视图 | 查看演进 |
| `markdown` | MD 格式导出 | 分享/备份 |
| `html` | HTML 格式 | 网页浏览 |
| `pdf` | PDF 格式 | 正式文档 |

## 💡 使用技巧

### 1. 上下文感知

```bash
# 加载后，记录自动使用该话题
/加载 AI 智能体
/记录 --summary "..."  # 无需重复话题名！
```

### 2. 空格分割

```bash
# 新语法 - 简洁
/加载 AI 智能体分类 summary
/搜索 智能体 10
/列表 7 10

# 旧语法 - 仍可用
/加载 --topic "AI" --format summary
/搜索 --query "智能体" --limit 10
```

### 3. 自然语言

```bash
# 直接说
"加载 AI 智能体话题"
"记一下，讨论了 XXX"
"搜索关于 AI 的讨论"
"列出最近的对话"
"总结一下"
"导出为 markdown"
```

## 🔧 故障排除

| 问题 | 解决方案 |
|------|----------|
| 找不到话题 | `/列表` 查看所有话题 |
| 上下文失效 | 重新 `/加载 话题名` |
| 记录失败 | 检查 `--topic` 和 `--summary` |
| 导出失败 | 确认话题存在 |
| 搜索无结果 | 尝试不同关键词 |
