# 🚀 快速开始 - 对话知识管理

## 📚 简化语法总览

所有指令都支持**空格分割**的简洁语法！

---

## 🎯 核心工作流

### 1️⃣ 加载话题 → 记录对话

```bash
# 加载话题（设置上下文）
/加载 AI 智能体分类

# 讨论完后直接记录（自动使用加载的话题！）
/记录 --summary "讨论了新的分类方法"
```

---

### 2️⃣ 搜索 → 查看 → 导出

```bash
# 搜索
/搜索 智能体

# 加载查看
/加载 AI 智能体分类 summary

# 导出
/导出 AI 智能体分类 markdown
```

---

### 3️⃣ 总结 → 分享

```bash
# 总结单个话题
/总结 AI 智能体分类

# 总结所有
/总结 all

# 列出最近
/列表 7 10
```

## 💡 语法对比

### 加载话题

| 新语法 ✅ | 旧语法 |
|----------|--------|
| `/加载 AI 智能体分类` | `/加载 --topic "AI 智能体分类"` |
| `/加载 AI summary` | `/加载 --topic "AI" --format summary` |

### 搜索

| 新语法 ✅ | 旧语法 |
|----------|--------|
| `/搜索 智能体` | `/搜索 --query "智能体"` |
| `/搜索 AI 10` | `/搜索 --query "AI" --limit 10` |

### 列表

| 新语法 ✅ | 旧语法 |
|----------|--------|
| `/列表` | `/列表 --days 7 --limit 10` |
| `/列表 30 10` | `/列表 --days 30 --limit 10` |

### 总结

| 新语法 ✅ | 旧语法 |
|----------|--------|
| `/总结 AI 智能体分类` | `/总结 --topic "AI 智能体分类"` |
| `/总结 all` | `/总结 --all` |

### 导出

| 新语法 ✅ | 旧语法 |
|----------|--------|
| `/导出 AI markdown` | `/导出 --topic "AI" --format markdown` |

## ⚡ 快捷技巧

### 上下文感知

```bash
# 加载后，记录会自动使用该话题
/加载 AI 智能体分类
/记录 --summary "..."  # 自动记录到 AI 智能体分类
```

### 格式简写

```bash
# 格式参数直接放在后面
/加载 AI summary      # summary 格式
/加载 AI full         # full 格式（默认）
/加载 AI timeline     # timeline 格式
```

### 数字参数

```bash
# 列表：天数 数量
/列表 7 10    # 最近 7 天，最多 10 条
/列表 30      # 最近 30 天
```

## 🎯 最佳实践

### ✅ 推荐做法

1. **使用新语法** - 简洁自然
2. **先加载后记录** - 利用上下文
3. **定期总结** - `/总结 all`
4. **导出备份** - `/导出 话题 markdown`

### ❌ 避免做法

1. ~~重复输入话题名~~ - 用上下文！
2. ~~复杂参数~~ - 用空格分割！
3. ~~不备份~~ - 定期导出！

## 🔧 故障排除

```bash

/加载 话题名

# 或手动指定
/记录 --topic "话题" --summary "摘要"
```
